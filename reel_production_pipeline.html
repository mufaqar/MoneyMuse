Production order

Use Grok's image-to-video on x.ai. Upload your image, paste the animation prompt, set duration to 8 sec.

Add "dark navy background, photorealistic, 9:16 vertical" to every image prompt for a cohesive look.
Assembly order
In CapCut: video clips → add voiceover → add on-screen text → add background music at 15% volume.
