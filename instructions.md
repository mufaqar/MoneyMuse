# SECTION 4: VIDEO SCRIPT

Write a complete YouTube video script based on a given topic.

Script rules:
- Write in a storytelling and conversational style similar to animated finance explainer channels
- Start with a strong psychological hook that makes viewers curious
- Use relatable real life situations about money and everyday financial decisions
- Explain financial concepts in a very simple way
- Include psychological insights about how people think about money
- Use examples, metaphors, and simple comparisons to make ideas easy to understand
- Gradually build curiosity and reveal the key idea step by step
- Avoid complicated financial jargon
- Keep the tone educational, engaging, and easy to follow

Script structure:
1. Hook - Start with a relatable situation or surprising idea that grabs attention
2. Problem setup - Explain the common belief or mistake people make about money
3. Core explanation - Break down the financial concept step by step using simple examples and psychology
4. Insight section - Reveal the deeper idea about how money, behavior, or wealth building actually works
5. Practical takeaway - Explain what viewers should understand or do differently
6. Ending - Close with a thought provoking conclusion that reinforces the key lesson

Additional instructions:
- Script length: 1800 to 2200 words
- Write in a natural voiceover friendly style
- Do not include scene directions or camera instructions
- Do not copy existing scripts but keep the same storytelling style and pacing
