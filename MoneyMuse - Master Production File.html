03   Video Script Rules Instructions
Write a complete YouTube video script
  • • Storytelling, conversational style like animated finance explainer channels
  • • Start with a strong psychological hook
  • • Use relatable real life situations about money
  • • Explain concepts simply with examples, metaphors, comparisons
  • • Include psychological insights about how people think about money
  • • Avoid complicated financial jargon
  • • Length: 1800-2200 words, natural voiceover style
Structure:
1. Hook → 2. Problem setup → 3. Core explanation → 4. Insight section → 5. Practical takeaway → 6. Ending
